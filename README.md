# Online Fraud Detection using ML

**Description**
This project detects fraudulent transactions using machine learning algorithms. It also handles class imbalance using SMOTE to improve prediction performance.

**Technologies Used**
- Python
- Scikit-learn
- Pandas
- NumPy
- SMOTE

**Features**
- Classifies transactions as fraud or not fraud
- Handles imbalanced dataset using SMOTE
- Trained using supervised machine learning models

**Dataset**
- Contains transaction data with labeled fraud and non-fraud cases

**How to Run**
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python manage.py   (or your main file name)

**Output**
- The model predicts whether a transaction is fraudulent or not
- Accuracy: 99% accuracy using Random Forest
