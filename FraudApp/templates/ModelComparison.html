Rule-Based vs Hybrid Rule + ML Engine

📌 Explanation
Rule-based systems use fixed thresholds such as transaction limits and balance checks. These systems fail when fraud patterns change.

The Hybrid Engine integrates:
  • Rule-based fraud indicators
  • Machine Learning probability scores
This improves adaptability, reduces false positives, and enhances fraud detection accuracy.
⚙️ Hybrid Risk Score Formula
Final Risk Score = Rule Risk Score + (ML Probability × Weight)
